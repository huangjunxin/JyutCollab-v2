✦ JyutCollab AI Agent

全站方便快捷可用的 AI Agent 優化計劃

目標：把目前偏「詞條查詢助手」的 Agent 升級為能理解使用指南、調用歷史、協助複雜搜尋、引導編輯/審核流程、並可被審計的全站操作助手。

已落地基礎

全站側邊 Agent、對話歷史與審計落庫

AI 助手已從浮動 Slideover 改為全站右側 Panel，由 Header 按鈕開關；後端已新增 conversation/message/audit event 儲存，支援歷史對話與工具調用審計資料。

已全部落地

工具擴展與前端聯動

docs、歷史、進階篩選、頁面動作、審計查詢、寫入工作流均已工具化。Agent 可靠調用明確工具，前端透過 action 佇列執行本地操作，所有高風險動作走確認策略。

五、複雜搜尋與正則協助設計

目前前端已經有進階公式與 regex rows；最佳做法不是讓後端重寫一套不一致的搜尋語法，而是讓 Agent 生成同一份前端可套用的篩選狀態。

1. 理解意圖例如「找廣州話裡釋義提到動物、粵拼以 ng 開頭、狀態待審核」。
2. 轉換條件普通條件進入 query/dialect/status；複雜條件進入 formula + regexRows。
3. 先預覽Agent 說明將套用哪些條件，並可顯示匹配數。
4. 本地套用apply_entry_filters 更新表格狀態，保持用戶可撤銷。
自然語言建議生成說明
找與動物有關的詞 regexRows: [{ field: "theme", pattern: "動物|獸|禽|魚|蟲", flags: "i" }, { field: "definition", pattern: "動物|獸|禽|魚|蟲", flags: "i" }] 不應只用 query=動物，因為相關詞可能在分類或釋義中以更細詞出現。
找粵拼以 ang 結尾的詞 regexRows: [{ field: "phonetic", pattern: "ang[1-6]?\\b", flags: "i" }] 前端正則比簡單 jyutping contains 更適合。
找已發佈但釋義少於 5 個字的詞 formula: "status == '已發佈' && len(definition) < 5" 需要公式能力，不應只查 Mongo query。

六、使用指南 Q&A 設計

用戶最常問的不是詞條資料，而是「我現在應該怎麼操作」。因此 docs 工具應成為 Agent 的第一優先能力之一。

Docs 索引

啟動時或 build 時掃描 content/docs/*.md，提取 title、headings、正文純文本、slug。

輕量搜尋

第一版可用關鍵詞/標題/heading 權重，不必立即上向量庫。後續再加 embedding。

引用來源

回答「如何操作」時附上「可前往 使用指南 → xxx」和 local navigation action。

上下文路由

如果用戶在 entries 頁問「怎麼篩選」，優先搜 entries-table、views-filters-and-labels、keyboard-shortcuts。

八、實施注意事項

不要把所有能力塞進 Prompt。

Prompt 只能改善模型選工具的傾向，不能保證資料可讀、權限正確、結果可審計。全站 Agent 的核心應是「明確工具 + 明確風險 + 前端 action handler + 審計」。

  • 已完成基礎：全站右側 Agent Panel、對話持久化、歷史對話列表、封存、SSE conversationId/requestId、工具/確認/錯誤審計落庫。
  • 已完成功能擴展：docs 歷史工具、進階篩選編譯器、頁面上下文 provide/inject、本地 UI 動作（5 個工具）、審計時間軸面板、寫入工作流工具（草稿/送審/審核）、語義搜尋 fallback、系統提示優化。
  • 安全邊界：Docs、歷史、搜尋、審計查詢屬於 safe；本地 UI 操作屬於 local_ui；草稿/送審/審核維持 draft_write/editorial/admin 並要求確認。
  • 權限一致：歷史工具沿用 /api/histories 的 contributor/reviewer/admin 可見規則；寫入工具驗證方言權限與審核角色。
  • 資料最小化：page context 只傳目前頁、篩選、選中 id、可見行摘要，不傳 token/cookie/raw response。
  • 可觀測性：每次工具規劃、fallback、local action 和確認都寫入 agent_audit_events，審計面板可查閱。
  • UI 可撤銷：套用篩選時顯示可清除的提示橫幅，一鍵清除所有篩選回到之前狀態。
