# 鍵盤快捷鍵

鍵盤操作可以大幅加快批量編纂速度，尤其適合連續校對多條詞條或快速接受 AI 建議。

## 詞條表格快捷鍵

| 快捷鍵 | 作用 | 補充說明 |
|---|---|---|
| ↑ / ↓ / ← / → | 在儲存格之間移動 | 適合逐行或逐欄校對。 |
| Enter | 進入編輯，或確認目前編輯 | 在可編輯欄位最常用。 |
| Shift + Enter | 在多行欄位換行 | 適合長釋義、例句、備註。 |
| Tab | 接受目前欄位的可見 AI 建議，或移到下一欄 | 適用於釋義、分類、語域等 AI 建議；接受前先確認內容。 |
| Shift + Tab | 移到上一欄 | 用於反向檢查。 |
| Escape | 取消編輯、關閉提示或選單 | 發現輸入不對時先按 Escape。 |
| 直接輸入文字 | 在目前選中欄位開始編輯 | 適合像試算表一樣快速錄入。 |

## 搜尋同文檔快捷操作

| 操作 | 用途 |
|---|---|
| 點擊文檔頁上方搜尋框 | 搜尋使用方法、欄位、快捷鍵或流程。 |
| 輸入多個關鍵詞 | 例如 `AI 例句`、`Tab 快捷鍵`、`審核 拒絕`。 |
| Escape | 關閉搜尋結果下拉框。 |
| 點擊搜尋結果 | 跳到對應文檔頁。 |

## 高效編纂節奏

一個常用節奏是：

1. 用篩選縮窄任務集合。
2. 用方向鍵逐格移動。
3. 用 Enter 進入需要修改的欄位。
4. 用 Tab 接受目前欄位合適的 AI 建議，或前進到下一欄。
5. 每完成一批後確認保存狀態。

## 使用快捷鍵時的注意事項

- 如果你在輸入法組字中，部分快捷鍵可能先被輸入法接管。
- 在多行欄位中，Enter 同 Shift + Enter 的效果可能因欄位類型不同而不同。
- 接受 AI 建議前請先讀完內容，不要只為了速度連續按 Tab。
- 批量編輯時，留意目前選中的方言點同狀態，避免改錯行。
