# 快速開始

JyutCollab 是一個用來協作編纂粵方言多音節詞語的平台。你可以在同一個工作環境裏記錄詞頭、方言點、音系形式、義項、例句、語域、主題分類、跨方言詞語組同審核狀態。

## 完成第一條詞條

如果你只是想先完成一次基本操作，可以照住以下流程做：

1. 在頂欄按 **詞條列表**。
2. 先用詞條列表上方的搜尋框搜尋詞頭，確認同一方言點下未有相同詞條。
3. 在詞條列表右上方按 **新建詞條**。
4. 選擇你有權限編輯的 **方言點**。
5. 輸入 **詞頭**；如有常見異寫或變體，可用 `主詞頭[變體一;變體二]` 格式一次輸入。
6. 補上 **擴展粵拼（J++）**；如同一詞條有多個讀音，可用分號分隔，例如 `gong2; gong3`。新建詞條填入詞頭後，如系統顯示泛粵典 / 字表讀音參考，可以採納作為草稿，但要按方言點核對。
7. 寫入第一個 **義項**，說明核心意思、語用限制同常見搭配。
8. 加入至少一個自然例句；例句應反映真實語境，而不只是直譯普通話句子。
9. 選擇語義 **主題分類**；不確定時可以先參考 AI 建議，再由人判斷。
10. 按 **保存草稿** 或 **保存**，確認沒有未保存提示。
11. 資料完整後按 **提交審核**，交由審核員檢查。

| 新增詞條 | 詞條列表右上 **新建詞條** | 建議先搜尋，避免重複建立同一方言點的詞條。 |

| 使用 Google 登錄 | 登錄頁按鈕 | 首次登錄引導設定方言點；已註冊用戶按電郵自動合併。詳見 [帳戶、角色同權限](/docs/account-and-permissions)。 |
| 連結/解除 Google 帳號 | 個人資料頁 | 查看連結狀態、連結新帳號或解除現有連結。 |

## 你通常會做的三件事

1. **新增或整理詞條**：為某個方言點建立詞條，補齊詞頭、讀音、釋義、例句同分類。
2. **比較跨方言材料**：用聚合視圖或詞語組視圖觀察同一詞語在不同方言點的形式、意義同語用差異。
3. **提交同審核成果**：貢獻者提交待審，審核員檢查語言學質素後通過或拒絕。

## 詞頭同擴展粵拼快速輸入

詞條表格會自動解析常用輸入格式：

- **詞頭變體**：用 `主詞頭[變體一;變體二]` 格式一次輸入。半形 `[]` 或全形 `［］` 都可以，分隔符支援半形/全形分號同逗號。
- **多個讀音**：用半形分號分隔，例如 `gong2; gong3`。表格平時顯示第一個讀音。

本系統使用 **擴展粵拼（Jyutping++，J++）** 記錄讀音：它以香港語言學學會粵拼為基底，為各粵語方言補充聲母、韻母、聲調標記。一般香港 / 廣州粵語可直接用普通粵拼。完整的變體格式、J++ 標記同讀音參考說明，請見 [詞條欄位](/docs/entry-fields)。

## 編輯已有詞條的流程

1. 在 **詞條列表** 用搜尋或篩選找到目標詞條。
2. 點擊要修改的表格欄位，或者用鍵盤選中後按 Enter。
3. 修改內容後保存。
4. 如果改動影響語義、讀音或跨方言對應，檢查詞語組中其他方言點是否也需要同步整理。
5. 如詞條已進入審核流程，避免只改格式而不處理審核意見。

## 使用 AI 建議的流程

1. 先輸入可靠的詞頭、方言點、讀音同初步釋義。
2. 觸發 AI 分類、釋義、語域或例句建議。
3. 檢查 AI 是否誤判語域、義項邊界、方言適用範圍或例句自然度。
4. 可按 Tab 接受目前欄位的可見建議，或者手動改寫後再保存。
5. 不要把未核實的 AI 內容直接提交審核。

## 常用快捷鍵

以下是最核心的鍵盤操作；完整列表同使用技巧請見 [鍵盤快捷鍵](/docs/keyboard-shortcuts)。

| 快捷鍵 | 作用 |
|---|---|
| ↑↓←→ | 在表格儲存格之間移動 |
| Enter | 進入編輯或確認目前編輯 |
| Tab | 接受 AI 建議，或移到右方欄位 |
| Escape | 關閉提示、取消編輯 |
| Shift + Enter | 在多行欄位換行 |

## 初學者最容易忽略的事

- **方言點不是地區備註**：它決定權限、重複檢查同跨方言比較方式。
- **詞形同詞語組要分清**：同一 lexeme 可以在不同方言點有不同詞形、讀音同語用限制。
- **例句要自然**：例句應展示本地話語境，不應只是為了解釋義項而硬造。
- **AI 不是審稿人**：AI 可以幫你起草，但不能替代方言知識同語料判斷。
- **保存狀態要留意**：看到未保存提示時，先保存再切換頁面或批量操作。
