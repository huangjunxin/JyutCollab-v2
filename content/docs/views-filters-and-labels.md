# 視圖、篩選同標示

當詞庫變大後，單靠逐頁翻看很難有效工作。視圖、篩選同標示功能可以幫你把任務轉化成清晰的小集合。

## 何時使用篩選

- 只想看自己有權限的方言點。
- 想處理某個狀態，例如草稿或待審。
- 想找某一類語義場的詞。
- 想檢查缺少讀音、例句或分類的詞條。
- 想找符合某些字形或格式規則的詞。

## 進階篩選

進階篩選適合更精細的資料清理。例如：

- 找出詞頭包含某個字的詞條。
- 找出釋義中可能含有未統一術語的詞條。
- 找出例句格式不一致的資料。
- 用多個條件同時縮窄範圍。

使用進階篩選時，建議先從少量條件開始，確認結果合理後再加入更多條件。

## 正則條件

如果你熟悉正則表達式，可以用它找更複雜的字串模式。適合：

- 批量檢查括號格式。
- 找出可能缺聲調的讀音。
- 找出包含特定標點或異體字的詞頭。
- 檢查例句結尾標點是否一致。

正則條件很強，但也容易過度匹配。批量操作前一定要抽查結果。

## 狀態標示

狀態標示幫助你快速辨認詞條處於哪個階段：

- 草稿：仍在編輯。
- 待審：需要審核員處理。
- 已通過：可作正式材料使用。
- 已拒絕：需要修改或暫不收錄。

## 規則標示

規則標示用來突出可能需要注意的資料，例如格式不一致、缺漏欄位或疑似重複。看到標示時不要只為了消除提示而修改；應先理解它指出的是資料問題、格式問題，還是審核流程問題。

## 建議整理策略

1. 選定一個明確任務，例如「補齊香港待審詞條的例句」。
2. 用方言點、狀態同欄位條件篩選。
3. 按表格逐條修改。
4. 每完成一小批就保存。
5. 最後清除篩選，抽查整體是否有副作用。
