# 首頁導覽

登入 JyutCollab 後，你會先看到儀表板首頁。這個頁面匯總了你的工作進度、全站數據同快捷入口，讓你不需要逐頁翻找就能掌握整體狀況。

## 頁面佈局

首頁由上至下分為四個區域：

1. **歡迎標題列** — 顯示你的帳號名稱，右側有「刷新」按鈕同「新建詞條」快捷按鈕。
2. **快捷操作區** — 四個方塊入口，直接前往常用頁面。
3. **統計卡片區** — 多張卡片展示你個人、審核、全站同貢獻者活躍數據。
4. **進階統計區**（往下捲動可見）— 包括你的貢獻趨勢圖表同填寫輔助成效分析。

## 快捷操作

| 入口 | 前往頁面 | 適用對象 |
|---|---|---|
| 瀏覽詞條 | [詞條列表](/entries) | 所有人 |
| 我的詞條 | [詞條列表](/entries?filter=mine)（已套用「我的」篩選） | 所有人 |
| 審核隊列 | [審核隊列](/review) | 審核員、管理員 |
| 編輯歷史 | [編輯歷史](/histories) | 所有人 |

## 統計卡片

### 我的詞條

顯示你建立的詞條按狀態分類的數量：總詞條、待審核、已發佈、已拒絕。點擊數字可快速導向對應篩選狀態的詞條列表。

### 審核數據

只有審核員或管理員可以看到。顯示目前待審核總數，以及你個人已完成的審核數量。點擊「待審核」數字可直接進入審核隊列。

### 全站數據

顯示整個平台的詞條統計：總詞條、已發佈、待審核、已拒絕。用來了解項目的整體規模同審核進度。

### 貢獻者活躍度

顯示實際貢獻者總數、近 7 天活躍人數同近 7 天新增貢獻量，幫助團隊了解近期編輯動態。

## 進階統計（往下捲動）

首頁往下捲動還有更多卡片：

- **我的貢獻統計（進階）**：以圖表展示你按方言點的貢獻分佈同近期編輯趨勢。
- **填寫輔助成效**：審核員同管理員可以查看 AI 建議同參考資料的採用情況。分為「AI 建議」同「參考資料」兩個分頁，統計各類輔助來源的總量、採用率同略過情況。詳見 [AI 輔助](./ai-assistance)。

## 頂欄導航

無論你在哪個頁面，頂欄始終顯示以下導航入口：

| 導航項 | 功能 |
|---|---|
| 首頁 | 返回儀表板首頁 |
| 詞條列表 | 進入主要編纂工作區，詳見 [詞條表格](./entries-table) |
| 審核隊列 | 查看待審詞條（僅審核員/管理員），詳見 [審核流程](./review-workflow) |
| 編輯歷史 | 追蹤所有詞條的修改記錄，詳見 [編輯歷史同還原](./history-and-revert) |
| 使用指南 | 開啟本指南文檔 |
| ✨ AI 助手 | 打開右側全站 AI 面板，詳見 [AI 輔助](./ai-assistance) |

## 個人資料同設定

點擊頂欄右側的用戶圖示或名稱，可以前往個人資料頁面。在那裏你可以：

- 查看同編輯個人資料。
- 管理 Google 帳號連結（詳見 [帳戶、角色同權限](./account-and-permissions)）。
- 查看你獲授權的方言點列表。
