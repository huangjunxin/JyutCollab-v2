<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| UI controls -> Excel-tool composables | User actions in toolbar/panels update local filter/rule/share state. |
| Expensive expressions -> browser main thread | Formula/regex/rule evaluation can affect responsiveness during typing/editing. |
| Keyboard events -> table navigation/editing | Global table handlers can accidentally intercept interactions inside controls. |
| Clipboard API -> fallback URL display | Copy failure exposes a selectable URL to the user without leaking extra state. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-02-01 | Denial of Service | filter/rule evaluation | mitigate | Cache unchanged formula parses and regex compilations where repeated; keep max-length and unsafe-regex validation in existing utility path. |
| T-04-02-02 | Tampering | keyboard handlers | mitigate | Guard table navigation handlers when focus is inside toolbar, panels, popovers, alerts, color picker, and fallback URL controls. |
| T-04-02-03 | Information Disclosure | share fallback URL | mitigate | Fallback URL remains selectable but contains only validated shared-view configuration; no entry/session/token serialization from Plan 04-01. |
| T-04-02-04 | Repudiation | hidden/no-op controls | mitigate | Disabled controls show readable reasons and invalid states use visible HK Traditional alerts/field errors. |
| T-04-02-05 | Elevation of Privilege | UI hardening changes | avoid | Do not add backend routes, saved-view persistence, bulk actions, new registries, or destructive operations. |
</threat_model>
